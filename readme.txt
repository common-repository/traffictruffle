== Description ==

An easy way to include the Traffic Truffle code throughout your site.

For more information visit:

http://www.traffictruffle.com

== Installation ==

1. Upload the traffictruffle directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the your Traffic Truffle code that we have given you to Admin > Settings > Traffic Truffle
